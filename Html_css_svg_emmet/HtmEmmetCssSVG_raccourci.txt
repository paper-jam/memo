-- ===== FLEXBOX
- fonctionnement à deux étages : le parent, les enfants
- dans le CSS parent : display : flex;
- flex-direction : row (par défaut) ou column

- flex-grow: 1;
- flex-shrink: 2: tu as le droit de te rétrécir 2 fois plus que les autres
- flex-basis: 0px ou em; on indique une valeur minimum
- flex-basis: auto : c'est la valeur initiale de l'élément, cela peut dépendre du texte qu'il contient
- justify-content: flex-end; (ou flex-start)
